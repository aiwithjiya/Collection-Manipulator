📂 Student Data Organizer
📖 Project Description
The Student Data Organizer is a Python-based menu-driven application developed to manage student records efficiently.
This project allows users to store, update, display, and remove student information using Python data structures and functions.
The program is simple, interactive, and beginner-friendly, making it suitable for learning basic Python programming concepts.
🎯 Objectives of the Project
• To organize student records systematically
• To perform CRUD operations using Python
• To understand menu-driven programming
• To practice Python functions and data structures
• To improve problem-solving and logical thinking skills
✨ Features
• Add new student details
• Display all student records
• Update existing student information
• Delete student records
• Display all subjects offered
• User-friendly menu-driven interface
• Stores multiple student records dynamically
🛠️ Technologies Used
• Python 3
• Jupyter Notebook
📚 Python Concepts Used
• Functions
• Lists
• Dictionaries
• Loops
• Conditional Statements
• User Input Handling
• String Manipulation
📂 Data Stored for Each Student
The program stores the following information:
• Student ID
• Student Name
• Age
• Grade
• Date of Birth
• Subjects
⚙️ Functionalities
1. Add Student
Allows the user to enter and store new student details.
2. Display All Students
Displays complete information of all stored students.
3. Update Student Information
Updates the details of an existing student using Student ID.
4. Delete Student
Removes a student record from the system.
5. Display Subjects Offered
Displays all unique subjects available in student records.
6. Exit
Terminates the program safely.
🔄 Program Workflow
1. The program starts with a welcome message.
2. A menu is displayed with different options.
3. The user selects an option.
4. The selected operation is performed.
5. The menu appears again until the user exits the program.
🧠 Data Structures Used
List
A list is used to store multiple student records.
Dictionary
Each student's information is stored in dictionary format using key-value pairs.
Example keys:
• id
• name
• age
• grade
• dob
• subjects
▶️ How to Run the Project

3. Copy and paste the program code
4. Run the notebook cell
5. Use the menu options to manage student records
